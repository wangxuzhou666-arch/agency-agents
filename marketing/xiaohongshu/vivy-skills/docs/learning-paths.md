## ⚡ Claude Code 插件使用

**这是一个 Claude Code 技能插件系统**，安装后技能会在对话中自动触发：

```bash
# 安装
git clone https://github.com/yourusername/xiaohongshu-skills.git
cd xiaohongshu-skills

# Claude Code 自动识别并加载所有技能
```

**使用方式**：
- 与 Claude 对话时，相关技能自动加载
- 或手动指定：`请使用 [skill-name] 技能`

详细说明：[installation-guide.md](installation-guide.md)

## 💡 学习建议

### 通用建议

1. **先学核心技能**
   - 每个路径前5个技能是基础
   - 打牢基础再进阶

2. **理论+实践结合**
   - 不要只学不做
   - 每学一个技能就去实践

3. **数据驱动优化**
   - 定期复盘数据
   - 根据数据调整方向

4. **查看技能文档**
   - 每个技能都有详细文档在 `skills/` 目录
   - 包含理论、案例、常见错误

### 新手特别建议

1. **不要贪多**
   - 先掌握1个细分领域
   - 不要什么都想做

2. **模仿优秀同行**
   - 找10个对标账号
   - 学习他们的内容模式

3. **坚持比完美重要**
   - 先完成再完美
   - 持续输出比质量更重要
