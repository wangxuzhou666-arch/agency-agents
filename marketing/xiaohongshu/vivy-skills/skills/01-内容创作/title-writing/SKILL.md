# Title Writing (标题撰写)

## Overview

Title writing is crafting compelling headlines that grab attention, spark curiosity, accurately represent content, and entice users to click - the critical first impression for content discovery on Xiaohongshu.

## When to Use

**Use when**:
- Writing titles for new posts
- Optimizing low-performing content
- A/B testing headline variants
- Improving click-through rates
- Making content more discoverable

**Do NOT use when**:
- Writing full content body (use copywriting-skills)
- Creating cover text (use cover-design)

## Core Pattern

**Before** (boring titles):
```
❌ "我的护肤routine"
❌ "好用的产品推荐"
❌ "今天分享穿搭"
❌ "关于xx的一些想法"
```

**After** (compelling titles):
```
✅ "学生党必看: 5个平价好物让你告别选择困难"
✅ "坚持这套护肤30天,变化惊呆了所有人"
✅ "没想到这个每天的习惯竟然毁了皮肤"
✅ "普通女孩如何逆袭? 我做对了这3件事"
```

**4 Title Frameworks**:
1. **Number + Promise** - "5 ways to..."
2. **Curiosity Gap** - "The secret nobody tells you..."
3. **Problem-Solution** - "Struggling with X? Here's the fix"
4. **Transformation** - "How I went from X to Y"

## Quick Reference

| Framework | Template | Example | Use For |
|-----------|----------|---------|---------|
| **Number** | "N个..." | "5个平价好物" | Lists, collections |
| **Promise** | "让你..." | "让你皮肤发光" | Benefit-driven |
| **Curiosity** | "没想到..." | "没想到这个...有用" | Surprising tips |
| **Negative** | "别再..." | "别再犯这个错误" | Problem-solving |
| **Before-After** | "如何..." | "如何从0到1万粉" | Transformation |

## Implementation

### Step 1: Identify Core Value

**Ask**: What's the ONE thing audience gets?

**Examples**:
- Tutorial → "学会..."
- Product → "发现..."
- Tips → "知道..."
- Story → "了解..."

### Step 2: Choose Title Framework

**Match content to framework**:
- Educational → Number or Promise
- Surprising → Curiosity
- Problem-solving → Negative or Problem-Solution
- Personal story → Before-After or Transformation

### Step 3: Draft 5-10 Variants

**Brainstorm multiple angles**:
```
Content: 平价防晒霜推荐

Variants:
1. "学生党平价防晒霜推荐" (basic)
2. "5个防晒霜误区,你中了几个?" (curiosity)
3. "这些百元防晒霜竟然比千元还好用" (surprising)
4. "别再花冤枉钱了,平价防晒霜才是真香" (negative)
5. "从过敏到最爱: 我和防晒霜的爱恨情仇" (story)
```

### Step 4: Optimize with Power Words

**Add emotional/trigger words**:
- **Urgency**: "必看", "立即", "现在"
- **Exclusivity**: "秘密", "没人告诉你", "终于"
- **Numbers**: "5个", "3步", "7天"
- **Results**: "惊呆", "逆袭", "变化"

### Step 5: Keep Under 20 Characters

**Xiaohongshu title limit**: ~20 Chinese characters

**Techniques**:
- Remove filler words
- Use concise phrasing
- Front-load important words
- Use symbols sparingly (emoji takes space)

### Step 6: Test and Iterate

**A/B test approach**:
1. Post with title variant A
2. Measure CTR (click-through rate)
3. Test variant B next time
4. Keep what works

## Common Mistakes

| Mistake | Fix |
|---------|-----|
| **Too long** | Keep under 20 characters |
| **Clickbait without substance** | Deliver on title's promise |
| **Vague** | Be specific about value |
| **No hook** | Start with number or curiosity |
| **Misleading** | Title must match content |

## Real-World Impact

**Great titles**: 2-3x more clicks
**Poor titles**: Great content gets <100 views

**Key metric**: Title is 50% of content success.
