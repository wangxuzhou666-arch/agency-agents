# Comment Strategy (评论策略)

## Overview

Comment strategy is the deliberate approach to engaging with Xiaohongshu audience through comments, using thoughtful replies, strategic questioning, and value-driven interactions to build relationships, boost algorithmic favor, and guide followers through the customer journey from casual observer to loyal customer.

## When to Use

**Use when**:
- Engaging with post comments
- Building community through discussion
- Answering follower questions
- Guiding potential customers to purchase
- Handling negative feedback
- Converting engagement to relationships

**Do NOT use when**:
- Comment is pure spam (delete or ignore)
- No value to add (generic response won't help)
- Trolling (don't feed trolls)
- Out of scope (off-topic)

## Core Pattern

**Before** (passive commenting):
```
❌ "Brief, generic replies"
❌ "Missing valuable opportunities"
❌ "Low engagement rate"
❌ "No relationship building"
❌ "Missed sales conversations"
```

**After** (strategic commenting):
```
✅ "Thoughtful, personalized replies"
✅ "Guided 25 followers to purchase"
✅ "Built strong community bonds"
✅ "Higher engagement and visibility"
✅ "Generated 18 new sales from comments"
```

**5 Comment Types**:
1. **Answer Questions** - Provide helpful information
2. **Show Empathy** - Connect emotionally
3. **Guide to Purchase** - Smooth sales funnel
4. **Start Discussion** - Drive engagement
5. **Address Concerns** - Handle objections

## Quick Reference

| Comment Type | Goal | Response Style | Success Metric |
|-------------|------|---------------|----------------|
| **Question Answer** | Be helpful | Detailed, personalized | Commenter replies with thanks |
| **Praise Reply** | Build connection | Grateful, engaged | Repeat engagement increases |
| **Objection Handling** | Convert concern | Understanding, solution | Commenter purchases |
| **Discussion Starter** | Boost engagement | Question-based | Thread expands |
| **Crisis Response** | De-escalate issues | Empathetic, professional | Issue resolved |
