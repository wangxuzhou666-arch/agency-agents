## Common Mistakes

| Mistake | Why Happens | Fix |
|---------|-------------|-----|
| Analysis paralysis | Overwhelming data | Focus on 3-5 key insights per audit |
| Comparing to wrong competitors | Seems logical | Compare to similar size, not market leaders |
| Ignoring qualitative data | Focus on numbers | Analyze comment sentiment, content themes |
| Acting on every trend | FOMO | Test trends before full commitment |
| One-time audits | No ongoing monitoring | Schedule monthly reviews, track trends |
| Benchmarking vanity metrics | Impressive numbers | Focus on actionable metrics (engagement, growth) |
| Ignoring seasonality | Short-term view | Compare YoY, account for seasonal patterns |
