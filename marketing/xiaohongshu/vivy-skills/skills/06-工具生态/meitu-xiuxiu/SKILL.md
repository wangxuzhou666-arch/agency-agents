# Meitu Xiuxiu (美图秀秀)

## Overview

Meitu Xiuxiu is a comprehensive mobile photo editing application that provides powerful yet user-friendly tools for enhancing, retouching, and stylizing images, making it ideal for Xiaohongshu creators who need to produce visually appealing content quickly without professional photography equipment or extensive editing skills.

## When to Use

**Use when**:
- Editing product photos for Xiaohongshu posts
- Enhancing selfies and portraits
- Creating before/after comparisons
- Adding text and stickers to images
- Retouching skin and facial features
- Creating collages and image layouts
- Batch editing multiple photos

**Do NOT use when**:
- Need professional-grade editing (use Photoshop)
- Complex compositing needed
- Video editing required (use video editing tools)
- High-resolution print output needed

## Implementation

### Step 1: Master Basic Enhancements

**Quick Improvements**:
```
One-Touch Enhancement:

1. Smart Enhance
   Automatic Optimization:
   - Open Meitu Xiuxiu
   - Select photo from gallery
   - Tap "Enhance" button (magic wand icon)
   - Choose "Smart Enhance"
   - App analyzes and improves:
     * Brightness (adjusts automatically)
     * Contrast (improves depth)
     * Saturation (enhances colors)
     * Sharpness (clarifies details)

   Result: Instantly better photo
   Best for: Quick fixes, batch processing

   Example:
   "Product photo looks dull and yellow-ish
   → One-tap enhance
   → Brighter, cleaner colors
   → Ready to post in 5 seconds"

2. Adjust Individual Parameters
   Fine-Tune Manual Control:

   Brightness:
   - Drag slider right (+) to brighten
   - Drag slider left (-) to darken
   - Target: Well-lit, not washed out or too dark

   Contrast:
   - Increase (+) for more dramatic look
   - Decrease (-) for softer, flatter look
   - Target: Clear separation between light and dark

   Saturation:
   - Increase (+) for more vibrant colors
   - Decrease (-) for muted, subtle tones
   - Target: Natural-looking, not oversaturated

   Sharpness:
   - Increase (+) for crisp details
   - Decrease (-) for softer, dreamy look
   - Target: Clear but not harsh

   Warmth:
   - Increase (+) for warmer (yellow/orange)
   - Decrease (-) for cooler (blue)
   - Target: Natural skin tones

   Example Workflow:
   "Editing Xiaohongshu product photo:

   1. Import photo
   2. Smart enhance (auto improve)
   3. Manually adjust:
      - Brightness: +10 (brighten slightly)
      - Contrast: +5 (add depth)
      - Saturation: +8 (vibrant colors)
      - Sharpness: +3 (clarify product)
      - Warmth: -5 (cool tone, clean look)
   4. Compare original vs edited
   5. Save and export"

3. Crop and Rotate
   Perfect Composition:

   Crop Options:
   - Free (custom aspect ratio)
   - 1:1 (square - Xiaohongshu feed)
   - 3:4 (vertical - optimal for mobile)
   - 16:9 (horizontal - stories/banners)
   - 4:3 (standard photo)

   Crop Best Practices:
   ✅ Follow rule of thirds (subject off-center)
   ✅ Remove distractions (clean background)
   ✅ Focus on main subject (product/face)
   ✅ Leave breathing room (don't crop too tight)

   Rotate/Straighten:
   - Auto-straighten (detects horizon)
   - Manual rotate (precise angle)
   - Flip horizontal (mirror image)

   Example:
   "Product photo crooked:
   → Use straighten tool
   → Align to horizon
   → Crop to 3:4 (Xiaohongshu optimal)
   → Result: Professional-looking composition"

4. Perspective Correction
   Fix Camera Angles:

   Use Cases:
   - Product shots (parallel lines)
   - Architecture (straighten buildings)
   - Flat lays (correct distortion)

   Tools:
   - Horizontal perspective (fix left-right tilt)
   - Vertical perspective (fix up-down tilt)
   - Auto (detects and corrects)

   Example:
   "Flat lay product shot distorted:
   → Use perspective correction
   → Adjust vertical slider
   → Make lines parallel
   → Result: Professional flat lay"

5. Blur and Vignette
   Artistic Effects:

   Background Blur:
   - Simulate DSLR depth of field
   - Focus on main subject
   - Blur distracting background

   Intensity Levels:
   - Light: Subtle blur
   - Medium: Noticeable but natural
   - Heavy: Strong bokeh effect

   Vignette:
   - Darken corners (draws eye to center)
   - Lighten corners (reverse vignette)
   - Use sparingly (can look dated)

   Best Practice:
   "Product portrait:
   → Light background blur
   → Focus on product
   → Subtle vignette
   → Result: Product pops, professional look"
```

## Common Mistakes

| Mistake | Why Happens | Fix |
|---------|-------------|-----|
| Over-editing skin | Heavy filters look good initially | Use subtler settings (20-40%), maintain texture |
| Inconsistent filter use | Choose favorite filter | Create custom preset, use consistently |
| Too small text | Want text to be subtle | Make text minimum 15-20% of image width |
| Low contrast text | Subtle looks sophisticated | Use high contrast, add shadow/outline |
| Wrong aspect ratio | Don't know platform specs | Always use 3:4 for Xiaohongshu feed |
| Overusing stickers | Fun to add | Less is more, 1-3 stickers max |
| Ignoring composition | Focus on subject only | Use rule of thirds, crop intentionally |
