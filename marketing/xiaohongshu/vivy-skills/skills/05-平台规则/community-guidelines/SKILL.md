# 社区准则 (Community Guidelines)

## 概述

**社区准则的核心原则：尊重他人、积极互动、维护环境、共建美好社区。**

小红书是生活方式分享社区，社区准则不仅是为了避免违规，更是为了营造积极健康的社区氛围，让每个人都能在这里分享和发现生活的美好。

## 使用场景

**需要本技能的典型情况：**
- 需要了解社区规范
- 希望建立良好社区关系
- 需要引导粉丝互动
- 遇到社区冲突
- 需要维护社区氛围
- 建设粉丝社群

## 核心模式

### ❌ 负面互动
```
负面评论
↓
争论对骂
↓
环境恶化
↓
优质用户流失
```

### ✅ 正向互动
```
尊重他人
↓
积极互动
↓
营造氛围
↓
优质内容涌现
```

## 实施步骤

### 第1步：理解社区价值观

### 第2步：遵守互动规范

### 第3步：营造正向氛围

### 第4步：处理冲突

### 第5步：建设社区文化

## 常见错误

| 错误 | 后果 | 正确做法 |
|------|------|---------|
| 攻击性言论 | 被举报封禁 | 尊重他人，理性讨论 |
| 煽动对立 | 破坏社区氛围 | 积极正向，避免对立 |

## 关键指标

### 社区健康度

```markdown
**优秀水平：**
✓ 正面互动 > 90%
✓ 粉丝氛围积极
✓ 无社区冲突

**需要改进：**
✗ 负面互动频繁
✗ 粉丝冲突
✗ 氛围负面
```
