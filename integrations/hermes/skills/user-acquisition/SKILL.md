# 用户获取 (User Acquisition)

## 概述

**用户获取的核心原则：多渠道布局、精准定位、优化漏斗、持续测试。**

用户获取不是简单的"涨粉"，而是系统性地通过多种渠道吸引新用户，优化从接触到转化的完整漏斗，实现可持续的用户增长。成功的用户获取需要在获客成本和用户价值之间找到平衡。

## 使用场景

**需要本技能的典型情况：**
- 粉丝增长停滞或缓慢
- 需要快速扩大用户基数
- 启动新账号或新产品
- 需要降低获客成本
- 想要优化转化漏斗
- 多渠道获客策略设计

**获客渠道分类：**
- **自然增长**：内容推荐、搜索、社交分享
- **付费获客**：广告投放、DOU+、KOL合作
- **合作获客**：跨界合作、账号互推
- **私域裂变**：推荐奖励、裂变活动
- **线下获客**：活动、展会、门店

## 核心模式

### ❌ 单一渠道粗放式
```
依赖单一获客方式
↓
粗放式投放
↓
成本高，效果差
↓
增长不可持续
```

### ✅ 多渠道系统化
```
多渠道布局
↓
精准定位目标用户
↓
优化转化漏斗
↓
数据驱动优化
↓
持续降低成本
↓
可持续增长
```

**关键差异：**
- 单一渠道：依赖一个来源，风险高，成本高
- 多渠道：分散风险，优化成本，可持续

### 第4步：优化获客成本

**核心逻辑：获客不是目的，目的是以合理成本获取高质量用户。**

#### 4.1 成本分析

```markdown
**获客成本计算：**

**单渠道获客成本：**
```
CAC（获客成本）= 总投入 / 获客数量

示例：
DOU+投放：
- 投放：1000元
- 获客：200粉丝
- CAC = 1000/200 = 5元/粉

内容制作：
- 投入：5小时×100元/小时 = 500元
- 获客：300粉丝
- CAC = 500/300 = 1.67元/粉
```

**综合获客成本：**
```
总投入 = 所有渠道投入总和
总获客 = 所有渠道获客总和
综合CAC = 总投入 / 总获客

示例：
- 自然获客：0元，500粉
- DOU+：1000元，200粉
- KOL合作：3000元，150粉
- 总投入：4000元
- 总获客：850粉
- 综合CAC = 4000/850 = 4.71元/粉
```
```

## 常见错误

| 错误 | 后果 | 正确做法 |
|------|------|---------|
| 只关注获客数量不关注质量 | 粉丝多但价值低，转化差 | 质量>数量，精准获客 |
| 过度依赖单一渠道 | 风险高，成本高 | 多渠道布局，分散风险 |
| 忽视获客成本 | CAC过高，无法盈利 | 监控CAC，优化ROI |
| 只关注新用户不维护老用户 | 获客流失抵消增长 | 新老并重，留存优先 |
| 频繁更换策略 | 无法验证效果 | 给策略时间验证 |
| 盲目追求涨粉速度 | 获客质量下降，成本飙升 | 平衡速度与质量 |
| 不做A/B测试 | 错失优化机会 | 持续测试，数据驱动 |
| 忽视漏斗优化 | 获客成本低但转化差 | 全漏斗优化 |

## 真实案例

### 案例1：新账号低成本启动

**背景：**
- 新账号，0粉丝
- 预算有限，月预算<2000元

**策略：**
```
第1月：建立基础
- 自然获客：80%
  - 每周5篇高质量内容
  - SEO优化
  - 互动运营

- 付费辅助：20%
  - DOU+：500元/月
  - 加热优质内容

结果：
- 获客：800粉丝
- CAC：0.63元/粉
- 自然获客占比：75%

第2-3月：加速增长
- 自然获客：60%
- 付费获客：30%
- 合作获客：10%

结果：
- 月获客：2000-3000粉
- 综合CAC：2-3元/粉
- 粉丝质量：高

第4-6月：规模化
- 自然：40%
- 付费：40%
- 合作：20%

结果：
- 月获客：5000-8000粉
- 综合CAC：4-5元/粉
- 粉丝：2.5万
```

**成功关键：**
- 优先自然获客（成本低）
- 付费放大优质内容
- 持续优化内容质量

### 案例2：降低获客成本50%

**问题：**
- 综合CAC：8元/粉
- 过度依赖付费（70%）

**分析：**
```
渠道数据：
- 自然获客：CAC 1元，占比20%
- DOU+：CAC 6元，占比50%
- KOL：CAC 20元，占比20%
- 其他：CAC 10元，占比10%

问题：
1. 自然获客占比太低
2. 付费获客成本偏高
```

**优化：**
```
第1步：提升自然获客
- 内容质量提升
- SEO优化
- 社交分享激励

结果：
- 自然获客：20%→40%
- 自然CAC：1元（不变）

第2步：优化付费获客
- DOU+精准定向
- 素材A/B测试
- 出价优化

结果：
- DOU+ CAC：6元→4元

第3步：淘汰低效渠道
- 停止KOL合作（CAC太高）
- 集中资源高效渠道

结果：
- 综合CAC：8元→4元（-50%）
```
